Comb¶

Overview¶

The Comb node is used to apply the shape of a Groom to Fibres or Particles. When applied to Fibres they will be deformed to follow the shape and direction of the Groom, when applied to Particles the N attribute ( by default ) will be set to an average of the direction of the Groom. If the Comb Attribute is supplied Yeti will actually create/modify the attribute and set to an average of the direction of the Groom.

Inputs¶

Input Description
0 Data Flow Fibres or Particles
1 Strands Groom/Guide Curves

Comb¶

Parameter Description
Weight an overall weight of the effect.
Weight Random a means of randomizing the effect.
Spread used to fan the fur out along the direction of the comb.
Comb Attribute by default comb will actually comb fibres and set $N on points, this can be used to specify some other attribute to affect.
